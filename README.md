# MyBike Landing Page

A responsive landing page for MyBike — a modern electric bicycle brand. The page showcases bike models, highlights key features like auto-unlock, integrated lights, and hydraulic disc brakes, and provides a contact form for potential customers.

## [DEMO LINK](https://artemstadnik.github.io/mybike-landing/)

## Design Reference

[Figma Design - MyBike](https://www.figma.com/file/NZQAIydtHo5QkINyGLHNcq/SUSPENDED-MyBike-landing-page)

## Technologies Used

- **HTML5** — semantic markup with BEM methodology
- **SCSS** — modular styles with variables, mixins, and responsive breakpoints
- **JavaScript** — deferred script for interactive features
- **Parcel** — zero-configuration bundler with SCSS support
- **Google Fonts** — Poppins font family
- **ESLint / Stylelint / LintHTML** — code quality and linting

## Getting Started

To set up the project locally:

1. Clone the repository:

```bash
git clone https://github.com/artemstadnik/mybike-landing.git
cd mybike-landing
```

2. Install dependencies:

```bash
npm install
```

3. Run the project locally:

```bash
npm start
```

The development server will start and open the page in your browser.

4. Build for production:

```bash
npm run build
```

## Features

- Fully responsive design (mobile 320px+, tablet 744px+, desktop 1260px+)
- Smooth scrolling navigation between sections
- Slide-out mobile menu with CSS-only toggle (no JS required)
- Bike comparison section with three product cards
- Detailed feature showcase with adaptive image layouts (wide/square)
- Contact form with validation and phone/email/address info
- Hover effects and transitions with consistent animation timing
- Optimized WebP images with lazy loading for fast page loads
- Font preloading for improved rendering performance
